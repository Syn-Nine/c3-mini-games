# Dungeon of the Phoenix

Released June 2025

### Controls:
- Move: Arrow Keys
- Attack: Bump
- Heal Potion: H
- Mana Potion: M
- Cast Magic: C

Sounds and music by Syn9.

Art by Syn9 with some icons from:
https://dennisbusch-de.itch.io/0xdbs-gasp-8x8-1bpp-rpg-icons
